Reverse A String

  1 minutes  |    89 words

Create a function that reverses a string.

The first one I can think of:

def reverse(string: str) -> str:
  return string[::-1]

Let’s see the right way to do it

First thing first. Check the input. What if someone input a none string?

if not string or type(string) != str or len(string) < 2:
    return "error"
  • it has to exists.
  • it has to be str type.
  • it has to be longer than 2 characters(this statement must to be in the end, because len can’t be used objects other than str).
